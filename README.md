# Интерфейс проектирования и обучения работе с системой "Умный дом"

Дипломный проект - Frontend приложение для проектирования и обучения работе с системой умного дома.

## Технологический стек

- **React 18+** - UI библиотека
- **TypeScript** - типизация
- **Vite** - сборщик и dev-сервер
- **React Router v6** - маршрутизация
- **Feature-Sliced Design (FSD)** - архитектура проекта
- **SCSS Modules** - стилизация компонентов
- **shadcn/ui** - базовые UI компоненты
- **TanStack Query** - управление серверным состоянием
- **Zustand** - управление клиентским состоянием
- **react-hook-form + zod** - формы и валидация
- **dnd-kit** - drag & drop функциональность

## Структура проекта (FSD)

Проект организован по методологии Feature-Sliced Design:

```bash
# Запуск dev-сервера
npm run dev

# Сборка для production
npm run build

# Предпросмотр production сборки
npm run preview

# Линтинг
npm run lint

# Проверка ошибок TS
npm run tsc

# Проверка всего: типы + линтер + форматирование
npm run check
```

## Архитектурные решения

### Feature-Sliced Design (FSD)

FSD используется для организации кода по слоям:
- **app** - инициализация приложения, провайдеры
- **pages** - страницы приложения (композиция виджетов)
- **widgets** - крупные составные блоки UI
- **features** - бизнес-фичи (добавление устройств, настройка и т.д.)
- **entities** - бизнес-сущности с их типами и логикой
- **shared** - переиспользуемый код (UI компоненты, утилиты)
